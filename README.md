# React Template

This repository contains the React project template I use the most often.

## Getting Started

### Project setup

First install the npm packages:

```bash
npm install
```
Then, run the development server:

```bash
npm run dev
```

## Technologies Used

Here is the list of default packages comes with the template:

- Vite
- React
- React DOM
- React Redux
- React Router DOM
- @reduxjs/toolkit
- tailwindcss
